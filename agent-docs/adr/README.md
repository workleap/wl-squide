# Architecture Decision Records

This folder contains Architecture Decision Records (ADRs) for significant technical choices.

## Format

Each ADR should follow:

```
# ADR-NNN: Title

## Status
Accepted | Superseded | Deprecated

## Context
What prompted the decision.

## Decision
What was decided.

## Consequences
What changed as a result.
```
